# 1️⃣ Operators — 🎟 Ticket Scanners of the Park

Operators evaluate conditions and help in decision-making.

### ✅ Comparison Operators

* `>=` Greater than or equal
* `>` Greater than
* `<` Less than
* `===` Strict equality

### ✅ Logical Operators

* `||` → OR (at least one condition true)
* `&&` → AND (both conditions must be true)
* `!` → NOT (reverse boolean value)

---

### 💻 Example

```js
let age = 16;
let hasVIPPass = false;

// Age check
console.log(age >= 18); // false

// OR condition
console.log(age >= 18 || hasVIPPass); // false

// AND condition
console.log(age >= 18 && hasVIPPass); // false
```

Now change:

```js
hasVIPPass = true;
```

```js
console.log(age >= 18 || hasVIPPass); // true
```

### 🔎 Summary

| Operator | Condition                              |   |                                          |
| -------- | -------------------------------------- | - | ---------------------------------------- |
| `        |                                        | ` | Returns true if **at least one** is true |
| `&&`     | Returns true only if **both** are true |   |                                          |

# 2️⃣ Conditions — 🎢 Ride Eligibility Gate

Used to control program flow.

### Syntax

```js
if (condition) {
   // executes if true
} else {
   // executes if false
}
```

---

### 💻 Example

```js
let height = 120;
let minHeight = 120;

if (height >= minHeight) {
  console.log("You can ride the roller coaster 🎢");
} else {
  console.log("Sorry, you are too short.");
}
```

---

### 🔥 Best Practice for Multiple Rides

Use **object-based configuration**:

```js
const rides = {
  rollerCoaster: 120,
  giantWheel: 100,
  freeFall: 150
};

let userHeight = 130;

for (let ride in rides) {
  if (userHeight >= rides[ride]) {
    console.log(`You can ride ${ride}`);
  } else {
    console.log(`You cannot ride ${ride}`);
  }
}
```

# 3️⃣ Loops — 🧹 Patrol Bots of the Park

Loops automate repetitive tasks.

### For-of Loop

```js
const bins = ["Zone A", "Zone B", "Zone C"];

for (let bin of bins) {
  console.log(`Checking bin in ${bin}`);
}
```

---

### ⚠️ Best Practices

* Always define a **stopping condition**
* Avoid infinite loops
* Prefer `for...of` for arrays
* Use `break` or `return` when needed

# 7️⃣ Promises — 🎫 Ticket Counter Queue

Promises handle asynchronous operations.

## States:

* Pending
* Resolved
* Rejected

---

### 💻 Example

```js
function buyTicket() {
  return new Promise((resolve, reject) => {
    let ticketsAvailable = true;

    setTimeout(() => {
      if (ticketsAvailable) {
        resolve("Ticket issued 🎟");
      } else {
        reject("No tickets left ❌");
      }
    }, 1000);
  });
}

buyTicket()
  .then(message => console.log(message))
  .catch(error => console.log(error));
```

---

# 8️⃣ Async / Await — 🚀 VIP Fast Lane

Cleaner way to handle promises.

---

```js
function getRideReady() {
  return new Promise(resolve => {
    setTimeout(() => resolve("Ride is ready 🎢"), 1000);
  });
}

async function startRide() {
  console.log("Preparing ride...");
  let message = await getRideReady();
  console.log(message);
  console.log("Enjoy the ride!");
}

startRide();
```

---

### 🧠 When to Use?

* API calls
* Database queries
* File reading
* Any delayed operation

# 9️⃣ Error Handling — 🛡 Safety Officers

Use `try...catch`

```js
try {
  let data = JSON.parse("Invalid JSON");
} catch (error) {
  console.log("Error parsing JSON:", error.message);
}
```

---

### Best Practices

* Always wrap risky code
* Log meaningful errors
* Avoid crashing the app

# 1️⃣1️⃣ Map — 🔐 Locker System

Key-value storage.

```js
const lockers = new Map();

lockers.set(101, "Handbag");
lockers.set(102, "Camera");

console.log(lockers.get(101)); // Handbag
console.log(lockers.size); // 2
```

---

### Map vs Object

| Feature          | Object         | Map      |
| ---------------- | -------------- | -------- |
| Key types        | String/Symbol  | Any type |
| Order preserved  | Not guaranteed | Yes      |
| Built-in methods | Limited        | Many     |

---

# 1️⃣2️⃣ Callbacks — 🎭 Stage Performer

Function passed into another function.

```js
function announce(callback) {
  console.log("Preparing stage...");
  callback();
}

announce(function() {
  console.log("Performer enters 🎤");
});
```

---

### Callback vs Event

| Callback            | Event              |
| ------------------- | ------------------ |
| Passed as argument  | Triggered by user  |
| Executes after task | Executes on action |

# 1️⃣4️⃣ Complete Farmer Help App 

## ✅ Full Working Example

```html
<!DOCTYPE html>
<html>
<head>
  <title>Farmer Help App</title>
  <style>
    body {
      font-family: Arial;
      padding: 20px;
    }
    #result {
      margin-top: 20px;
      font-weight: bold;
    }
  </style>
</head>
<body>

<h2>Farmer Help App 🌾</h2>
<p>Enter crop name:</p>

<input type="text" id="cropInput">
<button id="checkBtn">Check Crop</button>

<div id="result"></div>

<script>

const cropDatabase = {
  wheat: {
    season: "Rabi",
    tips: ["Sow in November", "Cool climate required"]
  },
  rice: {
    season: "Kharif",
    tips: ["Needs standing water", "Warm climate"]
  },
  maize: {
    season: "Kharif",
    tips: ["Moderate irrigation", "Warm climate"]
  }
};

const validCrops = new Set(Object.keys(cropDatabase));

function fetchExtraAdvice(crop) {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      if (validCrops.has(crop)) {
        resolve("Do soil testing before sowing.");
      } else {
        reject("No extra advice available.");
      }
    }, 1000);
  });
}

document.getElementById("checkBtn")
  .addEventListener("click", async function() {

    const crop = document
      .getElementById("cropInput")
      .value
      .toLowerCase();

    const resultBox = document.getElementById("result");

    if (!validCrops.has(crop)) {
      resultBox.innerHTML = "❌ Invalid Crop";
      return;
    }

    try {
      const data = cropDatabase[crop];
      const extraAdvice = await fetchExtraAdvice(crop);

      resultBox.innerHTML = `
        ✅ Valid Crop <br>
        Season: ${data.season} <br>
        Tips: ${data.tips.join(", ")} <br>
        Extra: ${extraAdvice}
      `;
    } catch (error) {
      resultBox.innerHTML = error;
    }
});

</script>
</body>
</html>
```
